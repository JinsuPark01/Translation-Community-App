## 🌐 Translation Community App

🔗 Original Repository
https://github.com/Android-Programming-Classroom-Project/FrontEnd

👨‍💻 Role  
Android Client Development

### 📌 Overview
대학 커뮤니티 서비스에 게시글 번역 기능을 결합한 안드로이드 애플리케이션.

### 🛠 Tech Stack
- Android (Kotlin)
- ML Kit Translation
- Android Resource Localization

### ⚙️ Key Features
- ML Kit 번역 모델을 활용한 게시글 번역 기능 구현
- 번역된 게시글 앱 내 렌더링
- Android 리소스 기반 UI 다국어 지원

### 💡 Implementation Highlights
- ML Kit Translation API 기반 온디바이스 번역 처리
- Android 리소스 분리(values, values-en)를 활용한 UI 언어 전환 구조 구현
